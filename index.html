Privacy Policy

Last updated: July 3, 2026

This Privacy Policy describes how this application ("Social Media AI Agent") handles data when connecting to Facebook to automate weather-forecast posts.

What the App Does

The App is an automation tool that:

  • Retrieves current weather data for a specified location from a third-party weather service.
  • Generates an image containing weather information using a third-party image-generation service.
  • Publishes the generated image and accompanying text as a post to a connected Facebook Page.

Data We Access

To perform the above, the App accesses the following through the Facebook API:

  • Page access permissions required to publish posts on behalf of a Facebook Page you manage.
  • Basic Page identification information (e.g., Page ID) needed to route the post to the correct Page.

The App does not access, collect, or store personal data belonging to Facebook users, Page followers, or visitors.

Data We Do Not Collect

The App does not collect, sell, rent, or share personal information, user contact details, or end-user location data.

Third-Party Services

The App relies on a weather data provider, an image-generation service, and Facebook's Graph API to function. Each has its own privacy policy.

Data Retention

The App does not maintain a database of personal user data. Weather data and generated images are used transiently to create each post.
